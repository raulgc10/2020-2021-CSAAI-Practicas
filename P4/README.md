 # Práctica 4
# Procesado de imágenes.

# Aplicación realizada en JS para el procesado de una imagen. 

# En la página aparecen 2 imágenes, a la izquierda, la original, y a la derecha la imagen modificada, a la cual pasamos un filtro.

# Hay 2 botones, uno para aplicar un filtro a la imagen y convertirla a escala de grises y otro para cambiar los colores de la imagen mediante unos deslizadores que nos permiten variar los valores RGB. Estos deslizadores tienen valores de 0-255.

# Además, estos deslizadores sólo sirven para el filtro de colores. 

# Al realizar un cambio en el filtro, éste se hace sobre la imagen original. Por tanto, si hemos cambiado los valores de los colores en el filtro de colores, si queremos cambiar a grises, la aplicación será sobre la imagen original y no sobre la que hemos modificado los colores.
